rss2dl_bubba
============

Rss2dl adjusted to support torrent download on the bubba mini server from excito

Configuration should be placed in a file called
conf_rss2dl.cnf

And the contents should be (this is my current):
[rss2dl]
passkey = XXX
user = tobias
timeLimitHours = 4
verbose = 0
episodeOnly = 1
